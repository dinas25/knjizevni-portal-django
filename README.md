# knjizevni-portal-django
Portal za dijeljenje i ocjenjivanje književnih djela

# Opis projekta
Ova web aplikacija razvijena je u sklopu projektnog zadatka za završni ispit kolegija Programiranje za web.
Cilj aplikacije je omogućiti korisnicima dijeljenje književnih djela, pisanje recenzija i ocjenjivanje sadržaja.

# Funkcionalnosti
- Registracija i prijava korisnika
- Unos književnih djela
- Prikaz i pretraga djela
- Pisanje recenzija i ocjenjivanje
- Uređivanje i brisanje unosa

# Tehnologije
- Django
- HTML / CSS
